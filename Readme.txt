Project Overview

This project performs exploratory data analysis (EDA) on an Amazon sales dataset using Python.
The goal is to understand sales patterns, product categories, trends, and insights using data visualization and basic statistics.

The analysis is done in a Jupyter Notebook, making it easy to follow step by step.
